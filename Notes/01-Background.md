We’re going to start with the basics of programming using a language called **Python**. It’s one of the most popular programming languages in the world.

If this is your first programming class, that’s great — we assume **no prior programming experience**.

Programming is the process of telling a computer what you want it to do in very precise detail. Python is one of the languages we can use to do that.

Python is one of the most widely used languages and is useful in many different fields. If you are a scientist, build websites, or work with data, there’s a good chance that knowing Python will be valuable.

Here’s what Python looks like:

```python
EARTH_CIRCUMFERENCE = 24901
PI = 3.14159

radius = input("How many miles is the radius of your planet? ")

# Circumference = 2πr
c = 2 * PI * float(radius)

print("Your planet's circumference is", c)
if c > EARTH_CIRCUMFERENCE:
	print("That's bigger than planet Earth!")
```

Now let’s try running it! We’re going to open something called a **REPL**. It stands for **Read, Eval, Print, Loop**. This is one way we can use the Python programming language.

A REPL lets us type Python commands one at a time and immediately see the results. Start by copying and pasting the Python code above.

We should get the following output:

```
How many miles is the radius of your planet? 12345
Your planet's circumference is 77565.8571
That's bigger than planet Earth!
```
