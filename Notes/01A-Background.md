# Assignment 01

## Introduction

We just learned how to use a REPL. That's a great start, but ideally we want to write programs that can be saved and run multiple times.

You have probably heard of **file types** before, such as `.zip` or `.txt`.

Python files use the `.py` **extension**. This suffix appears at the end of a filename and helps your computer understand what kind of file it is and how it should be handled.

In this assignment, you will create your first small Python program and get comfortable with:

1. Navigating the terminal
2. Creating and managing files
3. Running Python files

## Problem Statement

As a classic first exercise, you may need to install or use a UNIX-based system such as Linux or macOS. If you are using Windows, that is okay too — this assignment should still be doable.

Once you are set up, complete the following steps:

1. Look back at the notes for this section and find the command that displays output to the screen.
2. Use available resources such as Google, Stack Overflow, or the Python documentation to test how to print text to the screen.

Then create a folder called `01A` containing a single file named `hello.py`. Your file structure should look like this:

```text
└── 01A
    └── hello.py
```

Inside `hello.py`, write code that prints the following:

```text
Hello world, I'm ____ !
```

## Running Your Program

To run your file:

1. Make sure you are in the correct directory. If you type the UNIX command `ls`, you should see `hello.py` listed.
2. Run the file with:

   ```bash
   python hello.py
   ```

3. Check the output and make sure it matches what you expect.

If everything works, congratulations — you have written your very first real Python program!
